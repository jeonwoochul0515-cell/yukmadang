## 2. 디자인 컨셉 — "전통의 정성 × 모던한 감각"

AI가 만든 듯한 '템플릿형 깔끔함'을 의도적으로 피한다. 대신 다음을 추구한다.

### 2.1 비주얼 톤
- **참숯과 한지의 대비**: 깊은 차콜 배경 + 따뜻한 한지 베이지 + 숯불 오렌지 글로우
- **풀블리드 사진**: 보내주신 마케팅 컷을 화면 가득 채워 시각적 임팩트 극대화
- **타이포 강조**: 한글 세리프 (Noto Serif KR) + 한자/숫자에 굵은 디스플레이 폰트 혼용
- **여백의 미**: 한국 전통 미감 — 비대칭 레이아웃과 호흡 있는 여백

### 2.2 색상 토큰
| 토큰 | HEX | 용도 |
|---|---|---|
| `--charcoal` | `#15110D` | 메인 배경 (참숯) |
| `--ember` | `#E0531F` | 강조 색 (숯불 빛) |
| `--ember-glow` | `#F2A85B` | 부드러운 글로우 |
| `--hanji` | `#F2EBDA` | 라이트 배경 (한지) |
| `--clay` | `#8B5A3C` | 우디 액센트 |
| `--ink` | `#2A1F18` | 본문 텍스트 |
| `--bone` | `#E8DDC4` | 보조 베이지 |

### 2.3 타이포그래피
- **Display**: Noto Serif KR (Bold/Black) — 헤드라인·메뉴명
- **Body**: Pretendard Variable — 본문·UI
- **Accent**: 한자(肉) 큰 글씨 — 로고 옆 장식 요소

### 2.4 인터랙션
- Framer Motion: 스크롤 reveal, 패럴랙스, 메뉴 호버 줌
- 페이지 로딩 시: 어두운 화면에 숯불 빛이 번지듯 첫 화면 등장
- 메뉴 카드: 마우스 위치 따라 미세하게 기울어지는 3D 효과
- 사진 호버 시: 살짝 줌인 + 따뜻한 색온도 변화

## 4. 기술 스택

- Vite + React 18 + TypeScript
- Tailwind CSS v3 (브랜드 토큰 확장)
- Framer Motion (애니메이션)
- 글꼴: Pretendard CDN + Google Fonts Noto Serif KR
- 배포: Cloudflare Pages (GitHub 연동)
- 저장소: GitHub public — `yukmadang`

## 5. 카피라이팅 원칙

"AI가 쓴 듯한 무의미한 형용사"를 금지한다. 다음 룰을 따른다.

1. **감각 동사 사용**: "굽다 / 익다 / 배다 / 스미다" — 추상명사보다 동사
2. **숫자로 신뢰**: "당일도축" "650g" "16:00 오픈" "26.04.23 방문"
3. **고객 목소리 인용**: 실제 리뷰 문장을 그대로 (편집 없이)
4. **할머니 스토리 일관**: '앞마당 / 손주 / 정성 / 추억' 키워드 반복
5. **마침표 통일**: 한글 문장 끝은 `.` 또는 `,` — `:` 금지
